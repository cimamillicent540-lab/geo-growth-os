# Client Delivery Playbook

## Product packaging

Sell this as: AI Search Visibility Audit + Monthly GEO Growth System.

## Client onboarding data

- Brand name
- Website
- Target country
- Target language
- Product description
- Competitors
- Current offers
- Prohibited claims
- Compliance notes

## Gambling client checklist

- Confirm target jurisdictions
- Add responsible gaming language
- Avoid underage targeting
- Avoid guaranteed win language
- Avoid misleading bonus claims
- Add withdrawal, terms, bonus wagering, KYC, and legality FAQ tasks

## Crypto / exchange client checklist

- Add risk disclosure
- Avoid guaranteed returns
- Avoid “safe profit” language
- Add fees, withdrawals, custody, license, beginner education, copy-trading risk FAQ tasks

## Monthly service deliverables

- Weekly GEO visibility run
- Weekly report URL
- Competitor change summary
- 10-30 content tasks
- Landing page / FAQ recommendations
- Ad angle recommendations from high-intent queries
